# Our Travels 🌍

## 🇷🇺 Русский

**Our Travels** – комплексный, многоязычный проект о путешествиях, культуре и достопримечательностях. 
Цель проекта – сделать каждое путешествие **информативным и увлекательным**. 

### 🌐 Описание
- Многоязычный сайт: Русский 🇷🇺, Украинский 🇺🇦, Немецкий 🇩🇪 
- Динамический контент: шаблоны для стран, регионов, городов и достопримечательностей получают данные из JavaScript-объектов 
- Интерактивные элементы: поиск, сортировка, фильтры, интерактивные SVG-карты с ссылками, структурированная навигация
- Особенности: работа с большими данными, многоязычность, динамический рендеринг во фронтенде

### ⚙️ Используемые технологии
- React 
- Redux Toolkit 
- React Router 
- SCSS (Sass) 
- Vite 
- ESLint 
- React Icons 
- JavaScript (управление данными через объекты)

## 🌐 Деплой
https://our-travels.info/

## 🗂 Структура проекта
/src
  /components - React компоненты
  /datas - объекты с данными по странам, регионам, городам, достопримечательностям и событиям
  /pages - страницы стран, городов, достопримечательностей
  /styles - SCSS стили
  App.jsx
  main.jsx

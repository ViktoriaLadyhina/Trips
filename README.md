# Our Travels 🌍

## 🇩🇪 Deutsch

Our Travels ist eine mehrsprachige Webanwendung über Reisen, Kultur und Sehenswürdigkeiten. Das Projekt basiert auf einer strukturierten, hierarchischen Datenarchitektur und generiert Inhalte dynamisch für verschiedene Entitätsebenen (Länder, Regionen, Städte, Sehenswürdigkeiten).

### 🌐 Projektkonzept
Die Anwendung arbeitet mit einer datengetriebenen Architektur, bei der Inhalte als miteinander verbundene Entitäten organisiert sind. Auf dieser Grundlage werden Seiten und Navigation dynamisch im Frontend generiert.

### ⚙️ Implementierte Funktionen
* Mehrsprachigkeit (Deutsch / Ukrainisch / Russisch)
* Dynamische Generierung von Seiten basierend auf Datenstrukturen
* Hierarchische Datenstruktur (Länder → Regionen → Städte → Sehenswürdigkeiten)
* Suchfunktion innerhalb der Daten
* Filter- und Sortierlogik
* Interaktive SVG-Karten mit Navigation zwischen Entitäten
* Strukturierte Navigation innerhalb verknüpfter Inhalte

### 🧠 Architektur & Umsetzung
* Trennung von Daten- und Präsentationsschicht
* Routing-basiertes Rendering mit dynamischen Parametern
* Skalierbare Struktur zur Erweiterung neuer Entitäten ohne Anpassung der Kernlogik
* Zentrales Datenmanagement im Frontend

### 🚧 Aktuelle Weiterentwicklung (Fullstack Richtung)
Das Projekt wird aktuell zu einer Fullstack-Anwendung erweitert:

* Migration von statischen JavaScript-Daten in eine MySQL-Datenbank
* Entwicklung eines relationalen Datenmodells
* Aufbau einer REST API mit Node.js und Express
* Umstellung von statischen Datenstrukturen auf serverseitige Datenabfrage

### ⚙️ Verwendete Technologien
* React
* Redux Toolkit
* React Router
* SCSS (Sass)
* Vite
* JavaScript
* Node.js (in Entwicklung)
* Express.js (in Entwicklung)
* MySQL (in Entwicklung)

* Динамическая генерация страниц на основе данных
* Иерархическая структура сущностей (страны → регионы → города → достопримечательности)
* Поиск по данным проекта
